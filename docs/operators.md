## How this is computed

Every fork is grouped by GitHub owner. Skill packs and matched ecosystem projects are attributed to the same owner (packs via the pack's `author` field, ecosystem via the matched fork's owner). Footprint = clamped stars + 3 × novel-skill count + 5 × packs + 5 × ecosystem matches + 2 × additional forks. The cap on stars stops a single popular fork from dominating the ranking — engagement breadth matters as much as one viral asset.
